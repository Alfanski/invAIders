Pipeline Deep Dive

n8n: The orchestration engine

1

Webhook Receive

n8n webhook catches the forwarded Strava event with activity ID and athlete ID.

2

Token Resolution

Calls Convex HTTP endpoint to fetch a fresh Strava access token. Auto-refreshes.

3

Fetch Activity + Streams

10 data stream types (HR, pace, elevation, cadence, GPS...) from Strava API v3.

4

Downsample (Code Node)

30s rolling window, targets ~500 points. Computes HR drift. Builds scout prompt.

5

Two-Stage Gemini Analysis

Scout classifies workout. Coach produces structured coaching JSON.

6

Store & Complete

Analysis + streams persisted to Convex. Dashboard updates reactively.

Downsampling Logic
// Target ~500 points for any duration const windowSec = Math.max( 30, Math.ceil(durationSec / 500) ); // HR drift: aerobic decoupling signal const drift = (avgLast - avgFirst) / avgFirst;
Two-Stage Prompt Strategy
// Stage 1: Scout (temp 0.2, 2K tokens) workoutClassification, keyObservations analysisDirectives, relevantFrameworks // Stage 2: Coach (temp 0.3, 4K tokens) effortScore, executiveSummary positives, improvements splitAnalysis, nextSession
Why Two Stages?

Scout narrows context before coach writes. Prevents generic advice. Pattern detection separated from actionable coaching.

AI Coach Architecture

Not a chatbot. An agent.

The coach is a LangChain-powered agentic loop backed by Gemini 2.5 Flash. It doesn't hallucinate numbers -- it queries real workout data through 9 structured tools connected to Convex.

Technical Architecture

Full-stack reactive system

Frontend -- Next.js 14

App Router. Server-side auth. Three dashboard tabs. Custom glassmorphism design system. Recharts. Pure Tailwind v4.

Backend -- Convex

Reactive real-time subscriptions. Dashboard updates the instant pipeline writes. Strava tokens server-side only. Schema-validated mutations.

Orchestration -- n8n Cloud

Workflow-as-code: JSON in repo, deployed via REST API. Sync scripts for push/pull. Code nodes for downsampling + prompts.

AI Layer -- Gemini

Gemini 2.0 Flash for pipeline analysis (structured JSON, low temp). Gemini 2.5 Flash for interactive coach (LangChain tool calling).

Data Source -- Strava API v3

OAuth 2.0 with auto token refresh. Webhook subscription for real-time events. 10 stream types per workout. Rate-limit aware.
